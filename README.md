# Claude Token Tracker

Real-time countdown dashboard for Claude API usage windows, with persistent state, dynamic quotes, and dark mode support.

## Stack

- React 19 + TypeScript
- Vite 8 + Tailwind CSS v4
- Zustand (persist middleware)

## Dev

```bash
pnpm install
pnpm dev
```

## Deploy

Automatically deployed to GitHub Pages on push to `main`.
